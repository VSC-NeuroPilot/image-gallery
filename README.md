# NeuroPilot Image Gallery Repo

This is a living repo with assets used for NeuroPilot's image sidebar.

If you can't or don't want to read JSON, you can also view the file [here](https://raw.githack.com/VSC-NeuroPilot/image-gallery/main/sets.json) if your browser supports it.
